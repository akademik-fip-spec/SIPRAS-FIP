Tentang SIPRAS-FIP

SIPRAS-FIP disiapkan sebagai aplikasi berbasis Google Apps Script untuk membantu Fakultas Ilmu Pendidikan Universitas Negeri Padang mengelola informasi sarana dan prasarana secara digital.

Fondasi

Fitur Utama

Data Terpusat

Struktur aplikasi disiapkan untuk menyatukan data gedung, lokal, fasilitas, inventaris, pemeriksaan, dan jadwal.

Monitoring

Dashboard menjadi area kerja administrator untuk pemantauan dan pengembangan modul berikutnya.

Siap Berkembang

Layout, komponen, CSS, dan JavaScript dibuat modular agar mudah dipelihara pada tahap lanjutan.
